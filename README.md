# GonyWorld : 개인 웹 다이어리 플랫폼

## 소개

GonyWorld는 과거 Cyworld에서 영감을 받은 개인 웹 플랫폼입니다. 사용자는 다이어리를 유지하고, 사진을 저장하고, 게시판에 글을 작성하며, 방명록에 메시지를 남길 수 있습니다.


## 주요 기능
프로필 관리: 사용자는 개인 프로필을 업데이트하고 관리할 수 있습니다.
다이어리: 생각과 추억을 적을 수 있는 안전하고 개인적인 공간.
사진첩: 여행, 이벤트 또는 일상의 사진을 저장하고 볼 수 있습니다.
메시지 보드: 방문자와 소통하고, 그들이 메시지를 남기거나 게시물에 대한 생각을 공유하게 할 수 있습니다.
직관적인 UI: 고전적인 웹 시대에서 영감을 받았지만 현대적인 요소가 추가되었습니다.
안전한 인증: 안전한 인증 시스템을 통해 회원 가입, 로그인, 로그아웃을 할 수 있습니다.


## 기술

Python, Django, HTML, CSS
